2. Apache Maven 3.3.1

3. Spring Framework 4.1.6

4. Spring JMS API

5. Apache ActiveMQ 5.11.1

6. Apache Tomcat 8

7. Eclipse Luna 4.4

8. Log4j 1.2.17

9. SLF4J 1.7.12

10. JUnit 4.12

11. Mockito 1.10.19
